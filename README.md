# Deeeep.io Reef Client  

![](https://avatars.githubusercontent.com/u/111871339)

# What is DRC?  

Deeeep.io Reef Client is a project to create a minimalistic desktop client for v4 Deeeep.io

# Features  

- Asset Swapper  
- Custom Theme  
- Light Theme  
- Discord Rich Presence  
- Built-in Docassets  
- Evolution Tree  
- V3 UI
